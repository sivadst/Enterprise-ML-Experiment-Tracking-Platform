# ExperimentHub

Enterprise ML Experiment Tracking. A mini MLflow clone.

## Features
- **Projects, Experiments, and Runs** tracking.
- **Parameters, Metrics, and Artifacts** tracking.
- **Model Registry and Dataset Registry** with stage tagging.
- **Run Comparison** dashboard.
- **AI Reviews** engine that provides rule-based recommendations on training.
- Complete **Next.js frontend** and **FastAPI backend**.

## Quickstart

Run via Docker Compose:

```bash
docker-compose up --build
```

The frontend will be available at `http://localhost:3000` and the backend API at `http://localhost:8000/docs`. Sample database data is generated automatically.
